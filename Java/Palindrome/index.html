解析

可以将数字本身进行反转，如果反转后的数与原来的数想的即为回文数。这里需要注意，反转后可能会大于int.MAX导致溢出，我们可以考虑只反转数字的一半这样就避开了溢出的问题。
如何得知反转数的位数已经达到原始数的一半？
每次循环我们将原始数 / 10,反转数 * 10,当反转数大于原始数时跳出循环即处理一半数字。
另外负数肯定不是回文数，能被10整除的数除了0以外也都不是回文数。
